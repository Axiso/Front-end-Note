# =>


=>

ES6 允许使用“箭头”（=>）定义函数

使用注意的点:
* 箭头函数会改变函数内 this 的指向与上级作用域中的this指向保持一致。
* 不可用当做构造函数，也就是说不能用new调用。
* 不能在里边使用 arguments 对象，该对象在箭头函数中不存在，但还是可用使用剩余参数代替。（剩余参数）


箭头函数不能用new
